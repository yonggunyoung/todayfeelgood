# 🟦 광클대전 — 토스 미니앱 "신청→통과" 따라하기 (초보용)

> ⚠️ 솔직한 전제: **마지막 업로드와 제출은 당신의 토스 계정으로, 당신 컴퓨터에서** 해야 해요(토스 보안 규칙).
> 제가 못 누르는 부분이라, 아래 명령을 **그대로 복붙**하면 되게 다 준비해뒀습니다. 막히면 그 화면/에러를 저한테 그대로 주세요. 바로 고쳐드려요.
>
> 이 폴더(`toss-app/`)는 **기존 게임을 토스 업로드용으로 감싼 프로젝트**예요. 게임 원본은 한 폴더 위(`../`).

---

## 0. 준비물 (한 번만)
- **컴퓨터**(Windows/Mac)
- **Node.js LTS 설치**: https://nodejs.org → LTS 버튼 → 설치(다음다음).
- 설치 확인: 터미널(Windows=PowerShell)에서
  ```
  node -v
  ```
  버전(v20 등) 뜨면 OK.
- **yarn 켜기**(Node에 내장된 corepack 사용):
  ```
  corepack enable
  ```

## 1. 코드 받기
이미 GitHub에 있으니, 컴퓨터에 내려받아요:
```
git clone https://github.com/yonggunyoung/todayfeelgood.git
cd todayfeelgood/static-apps/gwangclick/toss-app
```
(git 없으면 GitHub 페이지 → Code → Download ZIP → 풀고 그 폴더로 이동)

## 2. 설치
```
yarn install
```

## 3. (선택) 미리보기 — 샌드박스
토스 "앱인토스 샌드박스" 앱으로 폰에서 미리 볼 수 있어요(필수 아님):
```
yarn dev
```
→ 터미널에 나오는 주소/QR을 샌드박스 앱에서 열기. (안 되면 건너뛰어도 제출 가능)

## 4. 빌드
```
yarn build
```
→ `dist/` 폴더가 생기고 그 안에 게임이 들어가면 성공.
> ❗에러가 나면 **에러 전체를 복사해서 저에게 주세요.** webpack/granite 설정만 손보면 됩니다.

## 5. 토스에 로그인 + 업로드
```
yarn deploy
```
- 처음엔 토스 로그인(브라우저) 창이 떠요 → **당신 토스 계정으로 로그인**.
- 업로드가 끝나면 콘솔에 우리 앱 빌드가 올라갑니다.
> `ait` 명령을 못 찾는다고 하면: `yarn deploy` 대신 `yarn dlx @apps-in-toss/cli deploy` 형태가 필요할 수 있어요. 그때도 에러 주시면 정확히 알려드릴게요.

## 6. 콘솔에서 앱 등록 + 제출
브라우저에서 **developers-apps-in-toss.toss.im → 콘솔**:
1. **워크스페이스** 만들기(처음 한 번).
2. **앱 등록(+등록하기)** → 우리 빌드 선택/연결.
3. **앱 정보 입력** (값은 전부 `../STORE.md`에 준비돼 있어요):
   - 이름: **광클대전**
   - 카테고리: **게임**
   - 한 줄 소개 / 상세 설명 / 키워드 → STORE.md 복붙
   - 아이콘: `../icon-512.png` 업로드
   - 스크린샷 2~4장(인트로/배틀/결과/지구본) → STORE.md 촬영 가이드
   - 연령등급: **전체 이용가**
   - 개인정보처리방침 URL: `https://ddukkit.com/gwangclick/privacy.html`
   - 문의 이메일: yonggunyoung@gmail.com
4. **제출** → 검수 4단계(운영·기능·디자인·보안), 보통 **2~3일**.

## 7. 검수 통과 팁 (미리 맞춰둠)
- **앱 내 완결**: 게임이 토스 안에서 끝까지 동작 ✅
- **외부 유도 금지**: 토스 안에선 "더 많은 앱" 링크 자동 숨김(코드 처리됨) ✅
- **개인정보**: privacy.html 고지 완비 ✅
- **광고**: **1차 제출은 광고 OFF**(`AD.enabled=false`) — 사업자 등록 전이라 이게 맞아요. 광고 없으니 광고 관련 반려 위험 0 ✅
- **연령/콘텐츠**: 폭력·사행성·선정성 없음(전체 이용가) ✅

---

## 8. 나중에: 광고로 수익화 (사업자 등록 후)
광고(보상형)는 **사업자 등록 + 계약/정산**이 있어야 켤 수 있어요.
1. 콘솔에서 사업자 등록(서류 심사 1~2일).
2. 콘솔에서 **보상형 광고 그룹 생성** → 발급된 **광고 그룹ID** 복사.
3. `../index.html`에서 `AD.tossAdGroupId='그ID'` + `AD.enabled=true`.
4. `toss-app/src/main.ts` 주석대로 SDK 전역 노출(광고 함수 연결).
5. 다시 `yarn build` → `yarn deploy` → 콘솔 재제출.

---

## 막히면
- **4번(빌드)·5번(deploy)에서 에러** = 거의 다 설정/명령 문제예요. 화면 그대로 저에게 주시면 토스 버전에 맞춰 `granite.config.ts`/`webpack.config.js`/명령을 고쳐드립니다.
- 콘솔 화면이 설명과 다르면, 그 화면 캡처 주세요. 토스 콘솔은 UI가 자주 바뀌어서 그때그때 맞춰드릴게요.
