# 🎵 "오늘 기분" — 제품 컨셉 & 리서치 정리

> 한 줄 정의: **이건 "기분일기 앱"이 아니다.** 하루 한 번 기분을 탭하면 *오늘의 노래*를 돌려받고, "오늘 대한민국 기분 67% 행복" 같은 **전국 실시간 집계**에 참여하며, 주·월 단위로 **나만의 플레이리스트 일기 카드**가 쌓이는 — **"광클대전의 따뜻한 버전"**. 일기 카테고리가 아니라 **SNS 인증/밈 카테고리**로 포지셔닝한다.

이 문서는 3개 리서치 에이전트(사용자 니즈 / 마케팅 기획 / 감사·리스크)와 음악·Wrapped 확장 논의를 한곳에 모은 단일 기획서다. (세션 휘발 방지용 — 코드보다 먼저 커밋.)

---

## 1. 핵심 컨셉

기분 1탭(5초, 글쓰기 없음) → **두 가지 보상 즉시**:
- **개인:** "오늘의 나 = 😀 산뜻" + 🔥streak + **오늘의 노래(유튜브 임베드)**
- **집단:** "지금 대한민국 ☀️ 행복 67%" 전국 게이지 + 우리 지역 순위

여기에 4개 레이어를 얹는다:
1. **음악 보상** — 기분 → 오늘의 노래 (재생은 유튜브/멜론으로 링크아웃, 오디오 호스팅 ❌)
2. **음악 성향 테스트(일회성 설문)** — 바이럴 유입구 (MBTI식 결과 카드)
3. **전국 기분/음악 집계** — "오늘 대한민국이 듣는 기분 노래 TOP" (멜론 차트엔 없는 감정 기반 전국 차트)
4. **주·월·연 Wrapped** — 나만의 플레이리스트 일기 카드 + 데이터 내보내기(본인 소유)

---

## 2. 리텐션 사다리 (이 제품의 척추)

혼자 쓰는 무드일기는 30일 리텐션 **6.1%**(아래 리서치). 단계별 보상으로 이 저주를 깬다:

| 주기 | 보상 | 효과 |
|---|---|---|
| 매일 | 기분 1탭 → 오늘의 노래 + 전국 게이지 | 즉시 보상 + "오늘 결과 보러 옴" 호기심 |
| 매주 | 이번 주 기분 분포 + 주간 플레이리스트 카드 | 주 카드 채우려 재방문 |
| 매달 | "6월의 나" + 이달의 플레이리스트(내보내기) | 투자 누적 → 이탈 방어 |
| 연말 | 올해의 감정 Wrapped | 스포티파이 Wrapped 공식(검증된 바이럴) |

---

## 3. 제품 원칙 5

1. **입력은 무조건 1탭 5초.** 글쓰기·이모지 30개 고르기 ❌ (이탈 원인).
2. **보상 이중화.** 탭 즉시 ⓐ개인(노래·카드·streak) + ⓑ집단(전국 게이지).
3. **공유 카드가 심장.** 인스타 스토리 9:16, "캡처해도 예쁨" = 유통. (디자인에 예산 몰빵)
4. **'오늘의 떡밥' 엔진.** 월요병·선데이블루·날씨연동·지역더비를 앱이 자동 카드화 → 매일 새 화제(콘텐츠 무한).
5. **익명·노출0·진영전 ON.** 얼굴·실명 없음(한국 정서). 광클의 지역전(서울 vs 부산)을 따뜻하게 이식.

---

## 4. MVP — 하나씩 채워나갈 순서

1. **기분 1탭 → 오늘의 노래(유튜브 임베드) + 내 기분 카드(이름·컬러)** ← 보상 있는 코어, 가장 먼저
2. 기분 **달력 + streak**(끊겨도 0리셋 ❌ → *프리즈* 1회)
3. **음악 성향 테스트(일회성 설문) → "음악 성향 카드"** ← 바이럴 유입구
4. **전국 집계**(Firebase) + "오늘의 기분 플레이리스트 TOP" + 콜드스타트 = **"전국 날씨" 메타포**
5. **공유 카드 9:16**(기분+노래+성향) — 카톡/인스타
6. **주·월 Wrapped 카드 + 데이터 내보내기**(JSON/CSV/곡 링크)
7. **앱인토스 입점**

> 빠른 첫 보상 주의: 주/월 카드는 신규 유저가 한 달을 기다려야 함 → "3일 미니 카드"나 *차오르는* 카드로 초반 공백 메우기.

---

## 5. 기술 방침 (변동비 0원 유지)

- **정적 PWA + Firebase 무료티어** (빌드 없음, GitHub Pages, 기존 앱들과 동일).
- **음악:** 곡명·아티스트 추천 + **유튜브 임베드/딥링크**(합법·무료). **냉비서의 유튜브 연동 코드 재사용.** 오디오 직접 호스팅/스트리밍 절대 ❌(라이선스·비용).
  - MVP 추천 엔진 = **손큐레이션 맵**(기분 × 성향 → 시드 플레이리스트). 검증 사례도 "엑셀 128곡"이 전부였음. 이후 유튜브 검색 API(무료 쿼터)/Spotify 추천 API로 고도화.
  - 자동 플레이리스트 생성(Spotify OAuth)은 **Phase 2**. 멜론은 공개 API 없어 곡별 딥링크만.
- **광클대전 자산 재사용:** `fb-config.js`(Firebase), `net.js`(실시간 집계), `gc-bots.js` 패턴, `geo.js`+`vendor/globe.gl`(→ 전국 기분 지도/히트맵), `i18n.js`, `toss.js`(앱인토스).
- **데이터 = 로컬 우선 + 사용자 소유.** 개인 기록은 localStorage, 서버엔 익명 집계만. 내보내기로 본인이 가져감. (↓ 프라이버시 전략 참조)

## 6. 리스크 & 가드레일

| 리스크 | 심각도 | 완화책 |
|---|---|---|
| **콜드스타트** (집계 0명 = 빈 깡통) | 🔴 높음 | "전국 날씨" 메타포(숫자 대신 ☀️🌧️) · 1개 커뮤니티(에브리타임 1학교/디시 1갤) 집중 시드 · 앱인토스로 모수 차용 |
| **봇으로 초기 채우기 유혹** | 🔴 높음 | 광클은 봇 OK였지만 **기분앱에서 가짜 감정데이터는 신뢰·윤리 타격이 큼** → 가짜 생성 ❌, "표본 적으면 숫자 대신 날씨 톤"으로 정직하게 |
| **리텐션 6.1%** (무드앱의 저주) | 🟠 중 | streak 의존 ❌ → "오늘 전국 결과 보러 옴" 호기심 루프 + 주/월 Wrapped 투자 루프 |
| **단발 바이럴 후 소멸**(Gas·포레스트형) | 🟠 중 | 떡밥 엔진으로 매일 새 화제 + 일일 코어가 받아줌 |
| **한국 노출 민감**(BeReal 실패) | 🟠 중 | 얼굴·실명 없는 익명 1탭 |
| ⏳ **PIPA 민감정보 · 의료기기 경계 · 미성년자 · 주의의무** | 🔬 정밀검토 미완 | **잠정 가드레일**: 감정데이터는 민감정보에 준해 취급(명시 동의·최소수집·익명집계) · "치료/진단/우울증 개선" 등 **의학적 표현 금지** · 위기 신호 시 전문기관(자살예방 1393 등) 안내 |

> ⚠️ 감사 에이전트의 **법률 정밀 리포트(PIPA 분류 확정)는 미완**. 출시 전 이 항목 확정 필요.

## 7. 프라이버시 전략 (= 차별점이자 법적 방패)

"본인이 데이터로 받아 활용"의 진짜 가치:
- **로컬 우선 + 사용자 소유 + 내보내기 가능** → 사용자 신뢰 ↑ ("내 감정 데이터를 내가 가져간다")
- 서버에 민감정보를 쌓지 않고 **익명 집계만** → PIPA 민감정보 부담 ↓
- 즉 Wrapped/내보내기 기능 하나가 **재방문 + 차별점 + 법적 리스크 완화**를 동시에 해결.

## 9. 바이럴 루프 · 채널 · 시즌 떡밥

**핵심 루프:** 기분 탭 → 개인+집단 보상 카드 → 카톡/인스타 공유 → 친구 "오늘 전국 기분 뭐임?" 진입 → 본인도 탭(집계 기여 → 소속감) → 반복.

**런치 채널 (콜드스타트 우회 순):**
1. **앱인토스** ★최우선 — 누적 5,100만, Web/PWA 입점, 계약 불필요, 출시 즉시 유입, 변동비 0과 정합
2. **에브리타임** — 1~2개 학교 "우리 학교 vs 옆 학교 기분 대결" 집중 시드
3. **인스타 스토리 / X** — 카드 공유(X가 감정 콘텐츠 궁합 최상)
4. **디시·더쿠·에펨** — "오늘 전국 우울 73% 역대급" 집계 캡처 → 지역전 밈
5. **틱톡/릴스** — 기분 캘린더·전국 타임랩스 데이터 시각화 숏폼

**시즌 떡밥:** 월요병 게이지 · 수능 D-day · 첫눈/벚꽃/장마 날씨연동 · 불금 vs 선데이블루 · 연말 결산 카드 · 서울 vs 부산 더비 · 사회이슈 감정 스파이크 · streak 마일스톤 배지.

## 11. 핵심 인사이트 (3개 리서치 교차 검증)

1. 차별점(전국집계·공유)은 일기앱 *안* 니즈가 아니라 *바깥* SNS/밈 문화에 있다 → 포지셔닝을 일기→밈으로.
2. 사랑받는 3요소 = **초간단 1탭 · 통계/달력 시각화 · 귀여운 마스코트**(하루콩 '콩'). 셋 다 계승, 마스코트는 공유 카드 주인공.
3. 바이럴은 "이름 붙은 예쁜 카드"에서 터지고, 죽음은 "혼자 쓰는 반복 없는 구조"에서 온다.
4. "오늘 실시간 전국 기분"은 빈 땅(공식 통계는 전부 비실시간).
5. 음악은 추가 기능이 아니라 **리텐션 약점을 푸는 보상**이자 **공유 artifact**.
