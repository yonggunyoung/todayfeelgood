# ✦ 글꾸미 (geulkkumi)

**텍스트를 멋글씨로, 사진·그림을 도트·문자 아트로 — 복사해서 인스타·카톡·디스코드·게임 어디든 붙여넣기.**
[snskeyboard.com/instafont](https://snskeyboard.com/instafont.php) 류의 특수문자 변환기를 출발점으로, **사진/그리기 → 아트**, **혼합 생성기**, **대형 특수문자·이모티콘·텍대 DB**, 그리고 **휴대폰 앱(PWA: 홈 화면 설치·앱 단축키·공유로 받아 변환)** 까지 한 앱에 담았습니다.

빌드 과정이 없는 **정적 웹앱(PWA)** 이라 어떤 정적 호스팅에든 올리면 바로 동작하고, 휴대폰에서는 홈 화면에 설치해 앱처럼 씁니다.

## 기능 (4기둥 + α)

| 화면 | 기능 |
|---|---|
| ✨ **멋글씨** | 입력 → 유니코드 스타일 실시간 변환을 **호환성 등급제**로: **✨ 안전셋(전각·볼드·동그라미 등)** 을 위에(자주 쓰는 순), **⚠️ 개성 스타일(프락투어·결합문자 등)** 은 경고와 함께 아래. **와이드 띄어쓰기**(전 플랫폼 안전)·**지옥체/팔다리체(채팅 뚫는 글씨)** 추가. **한글이 1급** — 프레임·사이 기호·원/괄호 한글·자모·초성·팔다리체. **글자수 카운터**(카톡 닉 20·인스타 150·X 280). **👁 실측 미리보기**(인스타·카톡·디스코드·X 목업 위에 결과 렌더 → 붙여넣기 전 확인)·**⭐ 맥락 추천**·**💾 슬롯 저장**. 탭하면 복사. |
| 🖼️ **사진아트** | 사진 업로드/드래그/붙여넣기 → **도트(브라유 2×4)·문자(ASCII)·하프블록·블록 음영·이모지** 아트. 결과 먼저, 세부(디더·명암·밝기)는 접이식. **PNG 저장**(브랜드 워터마크). |
| ✏️ **그리기** | 손으로 그린 그림을 도트·문자 아트로 변환. 펜 굵기·지우개·해상도. |
| 🎀 **꾸미기 DB·생성기** | **🛠 카오모지 메이커**(괄호·눈·입·팔·효과 부품 조합 → **425만+ 가지** 생성, 다 저장하지 않는 구조)·**🤣 밈 생성**(내 글자로 관짝밈·어몽어스·기가차드…)·**혼합 생성기**(글꼴＋끼우기＋프레임). **전역 검색**으로 특수문자(43카·**2,729**, 유니코드 블록 생성)·카오모지(68카·**1,304**)·구분선 46·**아스키 아트 갤러리**·**테마별 세트**(계절·우주·할로윈…)까지. 즐겨찾기. |
| 🗂️ **보관함** | 즐겨찾기 ★ · **슬롯 💾**(인스타 바이오·카톡 닉 등 용도별 저장) · 최근 복사 기록. |

> 설계는 **여러 역할 에이전트(기획·QA·디자이너·사용자 테스터·수익화 전략가)의 상반된 비평**을 종합해 다듬었습니다 — 한글 1급 처리, 전역 검색, 글자수/호환성 경고, 접근성(포커스·`aria`), 다크모드 일관성, 공유 URL·OG PNG 등이 그 결과입니다.

**휴대폰 편의 (PWA)**
- 홈 화면에 **설치**하면 전체화면 단독 앱 + 오프라인 동작(서비스워커 캐시).
- **앱 단축키**(길게 눌러 바로가기): 멋글씨 / 사진아트 / 그리기 / 꾸미기.
- **공유 타깃**: 다른 앱에서 텍스트를 *공유*하면 글꾸미가 열리며 변환 화면에 자동 입력 → 변환 → 복사.

## 채팅창에서 바로 — 확장 · 키보드 (형제 프로젝트)

복붙도 귀찮을 때, 입력하는 그 자리에서 쓰는 경로입니다. 모두 **메인 앱의 순수 엔진을 그대로 재사용**(무빌드).

- 💻 **PC 브라우저 확장** — 입력창에 `글꾸미 ✨` 배지 → 클릭→선택→**그 입력창에 즉석 삽입**. → [`geulkkumi-ext/`](../geulkkumi-ext)
- 📱 **안드로이드 커스텀 키보드** — 어떤 앱에서든 키보드 전환→**바로 입력**(Android Studio 빌드 스캐폴드). → [`geulkkumi-keyboard-android/`](../geulkkumi-keyboard-android)
- 🧭 **iOS 단축어·북마클릿·전 플랫폼 사용법** → [`MOBILE.md`](./MOBILE.md)

## 바로 띄우기 (GitHub Pages)

1. 이 브랜치를 `main`에 병합 → Settings → Pages → Source: `main` / `/(root)`
2. 발급 주소: `https://yonggunyoung.github.io/todayfeelgood/geulkkumi/`

Vercel·Netlify·Cloudflare Pages도 저장소 연결만 하면 끝(빌드 명령 없음, 출력 = 폴더 루트). 독립 도메인을 쓰면 `CNAME` 추가 + AdSense `ads.txt`를 도메인 루트에 두면 됩니다.

## 개발

```bash
cd geulkkumi
npm start   # http://localhost:8080 (python3 http.server)
npm test    # 변환·아트·꾸미기·한글·조합 코어 + 통합 테스트 (node --test, 36케이스)
```

## 수익화·배포 (전략가·디자이너 합의안)

도구 화면은 **광고 없이 깨끗하게**(경쟁 사이트와의 차별점) 유지하고, 즉효 레버부터 켭니다.

- **공유 바이럴 훅(반영됨)**: 공유 시 결과 + 앱 링크(UTM) 동봉, PNG 저장물 우하단 **브랜드 워터마크**, OG **PNG**(`og.png`, 1200×630 — 카톡/페북 스크래퍼가 SVG는 무시하므로 PNG로) + `og:url`·치수 메타 + **JSON-LD**·`sitemap.xml`로 SEO.
- **후원(1순위 수익)**: `js/main.js`의 `SUPPORT_URL`에 BuyMeACoffee/토스 링크만 넣으면 푸터에 ☕ 버튼 표시. 광고와 달리 심사·정책·개인정보 부담 0.
- **Pro(나중)**: 고해상 PNG(렌더 스케일 인자만 추가) · 프리미엄 프레임/팔레트 팩. 광고 제거형이 아니라 가치형.
- **AdSense(나중)**: 서브패스·SPA·복합 도메인이라 지금은 승인·단가 모두 약함 → 독립 도메인 + `blog/` 콘텐츠 페이지 확보 후 콘텐츠에 게재(`ads.txt`는 이미 도메인 루트에 있음). 도구 셸엔 넣지 않음.
- 모든 처리는 **클라이언트 단독**(사진/그림이 서버로 안 감) → 개인정보·서버비용 0, "광고 없이 빠름"이 마케팅 포인트.

OG 이미지를 새로 만들려면: `node tools/make-og.mjs` (순수 Node로 `og.png` 재생성).
