- 앱인토스 두 값은 **`js/config.js`의 `TOSS`** 에 넣으면 토스 안에서 보상형 광고·토스포인트 지급이 자동 활성화됩니다 (개별 운영엔 영향 없음). 절차: docs/10

- **애드센스 주의**: 계정은 1인 1계정 — 기존 계정이 있으면 새로 만들지 말고 [사이트] 메뉴에서
  이 서비스 도메인을 **추가**해 사이트별 심사를 받으면 됩니다. 다른 사이트에서 승인받은 코드를
  여기 붙이는 건 정책 위반(게재 제한)이고, `github.io` 같은 빌려 쓰는 주소로는 사이트 등록이
  안 되므로 **본인 소유 도메인 연결이 먼저**입니다.
- 쿠팡 ID 넣는 곳: 앱 → 관리자 모드 → AI 카드 → "쿠팡 파트너스 추적 ID"
- AdFit/AdSense/토스페이먼츠: 발급값을 저에게 주시면 광고 슬롯·결제 페이지에 제가 연결합니다

### 광고 동작 방식 (구현돼 있음 — 발급값만 꽂으면 됨)

- **보상형(핵심 수익)**: 무료 5회 소진 → AI 버튼 누른 순간에만 팝업 고지 →
  [광고 보고 계속하기] → **15초 완주해야 +1회** (바깥 탭으로 안 닫힘, 건너뛰면 충전 없음)
  → 완주 시 하던 작업(스캔 등)이 자동으로 이어짐. 미리 광고를 노출하지 않아 완주율·단가 방어.
- **배너 슬롯 3곳**: 홈 맨 아래(`id="ad-home"`) · 레시피 목록 아래(`id="ad-recipes"`) ·
  장보기 아래(`id="ad-shopping"`) — 지금은 하우스 광고가 돌고, AdFit 발급 후 이 자리에 교체.
  요리 진행·레시피 상세 화면에는 광고를 두지 않습니다(사용 경험 보호).
- **프리미엄 사용자**: 보상형 볼 일 없음(한도 통과) + 배너 슬롯 전체 미노출.

## 4. 관리자 모드 — 사용자에게는 보이지 않습니다

- 들어가기: **설정 맨 아래 "냉비서 v0.3 …" 문구를 4초 안에 7번 연속 탭** → PIN 입력
- 처음 한 번은 원하는 PIN을 정하는 화면이 나옵니다 → 여기 기록: `PIN: ______`
- 잠그기: 관리자 고급 카드 맨 아래 "관리자 모드 잠그기" 버튼
- 관리자에서만 보이는 것: 내 키(BYOK) 입력, 서버 주소, 유튜브 키, 쿠팡 ID, Firebase JSON, 가족코드 수동 입력

## 5. 사용자에게 보이는 선택지 (확인용)

- 무료: 모든 냉장고·레시피 기능 + AI 월 5회 (광고 시청으로 +1회씩, 하루 3회까지)
- AI 다 썼을 때: **[📺 광고 끝까지 보고 계속]** 또는 **[⭐ 프리미엄 알림받기]** — 이 둘뿐
- 로그인: **[구글로 시작하기]** 단 하나 (가족 공유는 로그인 뒤 선택)
- 🎮 미니게임 5종(냉장고 지키기·재료 매치·프레시 캐치·상식 퀴즈·외쳐!재료) +
  🅿 포인트(절약·게임 보상) + 🏆 랭킹(가족/전체). 첫 실행 시 사용법 튜토리얼.
  → 게임/포인트/광고 수익 설계와 정확한 수익성 계산은 **docs/10** 참고.
