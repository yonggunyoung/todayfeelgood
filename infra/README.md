# infra — 배포 · 라우팅

Oracle 무료 티어 기준 Phase 1 배포 설정.

## 구성
- `nginx/webapp.conf` — 서브경로 라우팅. **1단계는 `메인도메인/font`만 노출**(독립 도메인 분리는 Phase 3에서 301+canonical).
- `scripts/deploy.sh` — 프론트 빌드 + 엔진 venv 설치 + 기동 안내.
- `scripts/healthcheck.sh` — 엔진/프론트 생존 확인.

## 포트 규약
| 서비스 | 포트 |
|--------|------|
| 폰트 엔진(FastAPI) | 8000 |
| 폰트 프론트(Next) | 3001 |
| 메인 홈(Next, 후속) | 3000 |

## 주의 (무료 티어)
- 메모리/CPU가 작다. Next 빌드가 메모리를 많이 쓰면 스왑을 켜거나 사전 빌드 산출물 업로드를 고려.
- 폰트 엔진은 가볍지만(전통 방식, 비용 0), 동시 요청이 늘면 프로세스 수/타임아웃을 조절.
- 실제 도메인·TLS(certbot)는 도메인 연결 후 추가.
