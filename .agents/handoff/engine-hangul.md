# Engine 인수인계 — 한글 자모 조합(모아쓰기) `/hangul-compose`

> 한 줄 결론: 기본 자모 24자(자음14+모음10)를 그려 **text의 음절만** 규칙 affine으로 모아써 폰트(WOFF/TTF/…)를 만든다. 비AI·비용0. 정직: **"조합 글씨"**(획은 사용자 것, 모아쓰기는 합성).

## 엔드포인트
- `POST /hangul-compose` — 계약 `HangulComposeRequest{ jamo:[{char∈BASIC_JAMO, strokes}], text, refine, format }`.
- 응답: `HandwritingResponse` 형태 — `generatedBy:"handwriting"`, `glyphCount`=합성 음절 수, `fontFamily`(`MyHangul-<해시>`).
- 동시성: 별도 세마포어(`MAX_CONCURRENT_HANDWRITING`=2 재사용), 포화 시 503.
- 새 파일: `apps/font/engine/hangul_compose.py`. 테스트: `tests/test_hangul_compose.py`.

## 음절 분해 (S = code - 0xAC00)
- 초성 = S//588, 중성 = (S//28)%21, 종성 = S%28 (0=받침없음).
- 인덱스 → 호환 자모. 예: `값`→(ㄱ, ㅏ, ㅄ), `관`→(ㄱ, ㅘ, ㄴ), `안`→(ㅇ, ㅏ, ㄴ).
- 한글 아닌 문자(공백/영문/숫자)는 통과(글리프 미생성). space는 항상 cmap에 존재.

## 겹자모 근사 (전부 BASIC_JAMO 기본 자모로 분해)
- 쌍자음: ㄲ=ㄱ+ㄱ, ㄸ=ㄷ+ㄷ, ㅃ=ㅂ+ㅂ, ㅆ=ㅅ+ㅅ, ㅉ=ㅈ+ㅈ.
- 겹받침: ㄳ=ㄱ+ㅅ, ㄵ=ㄴ+ㅈ, ㄶ=ㄴ+ㅎ, ㄺ=ㄹ+ㄱ, ㄻ=ㄹ+ㅁ, ㄼ=ㄹ+ㅂ, ㄽ=ㄹ+ㅅ, ㄾ=ㄹ+ㅌ, ㄿ=ㄹ+ㅍ, ㅀ=ㄹ+ㅎ, ㅄ=ㅂ+ㅅ.
- 겹모음: ㅐ=ㅏ+ㅣ, ㅔ=ㅓ+ㅣ, ㅒ=ㅑ+ㅣ, ㅖ=ㅕ+ㅣ, ㅘ=ㅗ+ㅏ, ㅙ=ㅗ+ㅏ+ㅣ, ㅚ=ㅗ+ㅣ, ㅝ=ㅜ+ㅓ, ㅞ=ㅜ+ㅓ+ㅣ, ㅟ=ㅜ+ㅣ, ㅢ=ㅡ+ㅣ.

## 자모 글리프화 (handwriting.py 재사용)
- 그린 자모 획 → `_prepare_strokes_font_units`(메트릭 정렬 OFF, straighten=0) → `_stroke_outline`(곡선 외곽선) → 폴리곤 + 잉크 bbox(`_JamoInk`).
- 음절 배치 후 `_to_quad_contour`로 2차 베지어 컨투어 → 음절 1글리프(여러 컨투어).

## 모아쓰기(박스 분할, 폰트유닛 L=60..R=940, T=800..B=-160)
- 세로모음(ㅏㅓㅑㅕㅣ…): 초성 좌(0~0.60폭) + 중성 우.
- 가로모음(ㅗㅛㅜㅠㅡ): 초성 상(전폭) + 중성 하.
- 복합(ㅘㅝㅢ…): 초성 좌상 + 가로요소 좌하 + 세로요소 우.
- 받침 있으면 초+중 영역을 위로 압축(baseline 위 20%까지)하고 종성을 하단 전폭(겹받침은 좌우 분할).
- 각 칸: uniform scale(종횡비 보존, fill 0.86) + 중앙 정렬 affine.

## 가드 / graceful
- 입력 jamo는 BASIC_JAMO 24자만 허용(쌍자음 직접 그리기 금지 → 422). 점/글자수 상한은 손글씨 모델 재사용.
- 필요한 기본 자모가 없는 음절은 **스킵**(부분 합성). 합성 가능한 음절이 0이면 422.
- text 길이 ≤2000, 합성 음절 수 ≤ MAX_TOTAL_GLYPHS(120) — 무료티어 메모리 가드(전체 11,172 금지).
- 빈 jamo/빈 text → 422. refine은 pydantic 범위(REFINE_RANGES)로 clamp. 외부 API 없음(비용0).
- 재현성: head.modified 고정 → 같은 입력 동일 바이트.

## 테스트 / 검증
- `pytest -q` 전체 **97 passed**(신규 26 포함). 분해 정확성(값/관/안), 세로/가로/받침 각 글리프 생성, 겹자모(까/과/값) 근사, 그린 자모 부족 시 graceful, dedupe, 영문/공백 통과, woff/cmap 유효.
- uvicorn 라이브: 기본 자모(ㄱㄴㅇㅏㅗ) + `text="가나안 고"` → WOFF 생성·재오픈·cmap 4음절 확인 완료.

## 한계 (정직)
- **조합 티가 본질적으로 남는다.** 완벽한 8벌 조판 아님 — 초성/종성 자형이 모임유형별로 안 바뀌고, affine 스케일/이동으로만 근사. "읽히는 수준" 목표.
- 복합 모음/겹받침은 칸 분할이 거칠다(특히 ㅙㅞ 3요소, 겹받침 좌우). 받침 압축 비율은 고정 규칙표.
- 자모 잉크는 글로벌 라틴 가이드 정렬을 끄고 박스에 맞추므로, 그린 비율이 극단적이면 박스 중앙에 작게 들어갈 수 있음.
