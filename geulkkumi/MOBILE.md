# 글꾸미를 "채팅창에서 바로" 쓰는 법 (전 플랫폼)

복사→붙여넣기조차 귀찮을 때, 글꾸미를 **입력하는 그 자리에서** 쓰는 여러 경로입니다.
무엇이 *지금 바로 되는지*와 *네이티브 빌드가 필요한지*를 솔직하게 정리했습니다.

| 환경 | 방법 | 경험 | 지금 바로? | 위치 |
|---|---|---|---|---|
| **모든 기기** | 웹앱(PWA) 설치 + 공유로 받기 | 공유→변환→복사 | ✅ | `./` (이 앱) |
| **PC(크롬/엣지)** | 브라우저 확장 — 입력창 배지 | 배지 클릭→**즉석 삽입** | ✅ | [`../geulkkumi-ext`](../geulkkumi-ext) |
| **안드로이드** | 커스텀 키보드(IME) | 키보드 전환→**바로 입력** | ⚠ 빌드 필요 | [`../geulkkumi-keyboard-android`](../geulkkumi-keyboard-android) |
| **iOS** | 단축어(Shortcuts) | 공유→앱 열려 변환 | ✅(무설치) | 아래 레시피 |
| **iOS** | 커스텀 키보드 익스텐션 | 키보드로 바로 입력 | ⚠ Swift 필요 | (로드맵) |
| **모든 브라우저** | 북마클릿 | 선택→앱 열기 | ✅ | 아래 |

---

## 1. 웹앱(PWA) — 모든 기기, 지금

- 사이트 접속 → **홈 화면에 추가**(안드 Chrome 메뉴 / iOS Safari 공유→홈 화면에 추가) → 전체화면 앱 + 오프라인.
- **공유로 받기(안드/데스크톱)**: 다른 앱에서 텍스트 *공유* → 공유 메뉴에 **글꾸미** → 변환 화면에 자동 입력(`manifest.webmanifest`의 `share_target`).
- **앱 단축키**: 홈 아이콘 길게 눌러 멋글씨/사진아트/그리기/꾸미기 바로가기.

## 2. PC 브라우저 확장 — 즉석 삽입(권장)

[`geulkkumi-ext`](../geulkkumi-ext) 폴더를 크롬 `chrome://extensions → 개발자 모드 → 압축해제 로드`.
입력창을 클릭하면 `글꾸미 ✨` 배지가 떠서, 클릭→선택하면 **그 입력창에 바로 삽입**됩니다(막히면 클립보드 폴백). 설치/사용법은 해당 폴더 README 참고.

## 3. 안드로이드 커스텀 키보드 — 바로 입력

[`geulkkumi-keyboard-android`](../geulkkumi-keyboard-android)는 **Android Studio로 빌드**하는 IME 스캐폴드입니다.
키보드 본체가 메인 앱 엔진을 그대로 쓰고, 선택 시 `commitText`로 채팅창에 직접 입력(복붙 불필요). 빌드 절차는 해당 폴더 README 참고.

## 4. iOS 단축어(Shortcuts) — 무설치, 지금

키보드 익스텐션(네이티브) 없이 "공유→변환"을 만드는 법:

1. **단축어** 앱 → 새 단축어 → 동작 추가.
2. `공유 시트에서 받기`(입력: 텍스트) 켜기.
3. `URL` 동작: `https://yonggunyoung.github.io/todayfeelgood/geulkkumi/?text=` 뒤에 `[단축어 입력]`을 `URL 인코딩`해 이어붙이기.
   - (동작: 텍스트 → URL 인코딩 → 텍스트 결합)
4. `URL 열기` 또는 `Safari에서 열기`.
5. 단축어 이름을 "글꾸미로 꾸미기" → 공유 시트/홈 위젯/잠금화면에 추가.

이제 어떤 앱에서 텍스트를 **공유 → 글꾸미로 꾸미기** 하면 변환 화면이 열립니다(거기서 탭 복사). 자주 쓰는 스타일별로 단축어를 여러 개 만들면 "한 탭 변환"에 가깝게 됩니다.

## 5. 북마클릿 — 모든 브라우저(모바일 포함), 무설치

브라우저 즐겨찾기에 아래 주소를 저장하고, 글 쓰다가 텍스트를 선택한 뒤 즐겨찾기를 누르면 그 텍스트로 글꾸미가 열립니다.

```
javascript:(function(){var t=(window.getSelection?getSelection().toString():'')||'';location.href='https://yonggunyoung.github.io/todayfeelgood/geulkkumi/?text='+encodeURIComponent(t);})();
```

(모바일 사파리/크롬에서도 동작. 페이지를 떠나 앱으로 이동하는 방식이라, "그 자리 삽입"은 1·2·3번이 더 매끄럽습니다.)

## 복붙 호환 팁 (깨짐 줄이기)

- **항상 일반 텍스트로** 붙여넣기(서식 없이). 확장/키보드는 자동으로 일반 텍스트만 입력합니다.
- **⚠ 표시 스타일**(프락투어·결합문자·지옥체·네모/검은네모 등)은 인스타 이름칸·일부 안드로이드에서 □로 깨질 수 있어요. **닉네임엔 전각·볼드·동그라미** 같은 안전(✨) 스타일을 권장.
- **검색·해시태그**는 멋글씨를 평범한 글자로 정규화해 잡습니다. 계정 검색이 중요하면 이름엔 평문도 함께.
- **글자수**: 보이는 글자수와 실제 코드 길이가 달라요(특히 X/트위터는 일부 글자를 2자로 셈, 결합문자는 더 늘어남). 앱의 글자수 표시를 참고하세요.
