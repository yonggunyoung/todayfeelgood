# 팀 리뷰 종합 — "오늘 기분" 완성형 (기획·디자이너·감사·리뷰어)

> 4인 에이전트 리뷰(2026-06-23)를 종합한 결론 + 완성형 로드맵 + 후속 결정.
> 현 상태: **코드는 홈+결과 2화면(축약형)**, 나머지는 시안/목업. 아래는 "완성형"으로 가는 합의.

## 1. 4인 리포트 핵심
- **기획(PM):** 완성형 = 16화면. 빠진 핵심 = 온보딩·**음악 성향테스트(바이럴 유입구)**·설정·공유플로우·내보내기·전역 상태(로딩/빈/에러/오프라인). 네비 = 하단 4탭(홈·날씨·컬렉션·더보기) + 해시 라우터. **공유 버튼이 빈 함수**인 게 최대 실구현 갭.
- **감사(Audit):** "기능 전부 동시"는 1인 비현실. **MVP는 혼자서 보상 완결되는 코어.** 🚫 지도(globe.gl 1.8MB·세계지도라 부적합)·푸시알림(PIPA 토큰)·다크모드·결제스토어·Cloud Functions(유료) = **미루기/금지**. Firebase는 `gwangclick/net.js` 재사용(집계 increment·1일1회). **단일 출시 차단 = PIPA 미확정.**
- **디자이너:** 세련됨 = 위계×여백×일관×디테일. 모듈러 타입스케일(1.25)·8pt 간격·**2단 그림자**·무드-ink 대비 토큰·결과 위계교정(겹침 해결: head↔card 24px + 깊이 분리)·픽셀 지도·다크 토큰·프리미엄(천공/엠보스) — **정체성 토큰(코랄·종이·구름이·Cafe24)은 불변, 값·위계만 정교화.**
- **리뷰어(QA):** 치명 3 = ① 고정계약 토큰/클래스가 코드에 전무(스펙 stale) ② 폰트 문서(Jua)↔코드(Cafe24) 불일치 ③ 이모지(📲▶✓) 아이콘 사용. + 마스코트 5기분 실루엣 동일·왼쪽 쏠림, 위계 평평, 저대비. 카피 "구름이 보이스"로 개선표 제공. 완성 수용기준 제시.

## 2. 후속 결정 (D12~D15)
- **D12 — 정체성 단일화:** 마스코트=**구름이**, 디스플레이=**Cafe24Ssurround**로 확정. `DESIGN_SPEC.md`의 "기분이/Jua"는 stale → 갱신. (코드가 진실)
- **D13 — 토큰 단일출처:** 무드 5색은 **CSS 변수(`--happy` 등) 한 곳**만. `moods.js`·`mascot.js` 하드코딩 색 제거/참조화. 접근성용 `--{mood}-ink` 변형 추가(작은 글자용).
- **D14 — MVP 경계(감사 채택):** MVP = 온보딩 · 홈 · 결과(음악) · **공유카드(실제 생성)** · **음악 성향테스트** · 컬렉션/streak(프리즈) · 최소설정 · **개인정보방침** + 전역 상태. 전국집계는 "날씨 톤" 정적→Firebase 후속. 🚫 지도·다크·알림·결제·Functions는 2·3차.
- **D15 — 음악 링크 정확도:** `songs.js` 검색링크 → **곡별 YT/YT Music 딥링크**로 승급(D2 유지).
