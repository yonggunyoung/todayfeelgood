## 가드레일 (D8~D11 부속)
- 생성은 **Suno 제품 내**에서 (비공식 API 대량 자동생성 ❌)
- **가사 직접 작성** → 인간 저작물성 확보
- **무드로 추천 ⭕ / 특정 가수 스타일·목소리 모방 ❌**
- **"AI 생성" 표기**
- 음악은 **유료화 ❌** — 무료 미끼, 광고로 수익화
- ⏳ **PIPA 민감정보 정밀검토 미완** — 상업화 전 약관·법률 확정

## 증분 경계 (원칙3)
- **증분1(현재):** 기분 1탭 → Tier A 노래(링크아웃) → 내 기분 카드 + localStorage. 서버 0.
- 변경 반경 = `newapp/` 폴더 안. 냉비서·gwangclick·루트·워크플로 **무손상**.
- 디자인(preview.html)은 외부 작업 중 — 로직은 `js/*`에 분리되어 디자인 독립.
