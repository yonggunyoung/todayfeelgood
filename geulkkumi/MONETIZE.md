# 글꾸미 수익화 설계 (MONETIZE)

> 원칙: **도구 셸(만드는 화면)은 광고로 더럽히지 않는다.** 복붙 한 번이 핵심 경험인데
> 그 위에 광고를 얹으면 이탈한다. 돈은 *콘텐츠*와 *절제된 자리*, 그리고 *팬(후원·PRO)* 에서 번다.
> ddukkit.com 한 도메인으로 승인받으면 `/geulkkumi` 도 같은 사이트로 묶인다(아래 4항).

---

## 0. 한눈에 — 우선순위(노력 대비 수익)

| 순위 | 스트림 | 즉시성 | 필요한 것 | UX 비용 |
|---|---|---|---|---|
| 1 | **후원 버튼**(Buy Me a Coffee / 토스 송금) | 오늘 | 링크 1개 | 0 (푸터에 작게) |
| 2 | **콘텐츠 페이지 AdSense**(ddukkit 홈 ‘읽을거리’) | 승인 후 | 퍼블리셔 ID·`ads.txt` | 0 (도구 밖) |
| 3 | **도구 하단 1슬롯**(스크롤 끝, 인피드형) | 승인 후 | 슬롯 ID | 낮음 |
| 4 | **PNG 워터마크 → PRO 해제** | 1~2주 | 결제·플래그 | 낮음(무료는 유지) |
| 5 | **테마/스티커팩 IAP** | 나중 | 결제·디자인 | 0 (원하는 사람만) |
| 6 | **토스 미니앱 보상형 광고** | 입점 시 | 토스 입점(광클대전과 동일) | 0 (자발적 시청) |

권장 출발: **1 + 2 부터.** 트래픽이 의미 있게 붙은 뒤 3 → 4 순으로.

---

## 1. 후원(Tip) — 지금 당장, 마찰 0
- `index.html`에 이미 `#support-link`(후원하기) 자리 + `main.js`의 `SUPPORT_URL` 훅이 있다.
  링크만 넣으면 **푸터에 ‘☕ 후원하기’** 가 뜬다. (비어 있으면 자동 숨김 → 지금은 OFF.)
- 추천: **Buy Me a Coffee**(해외/카드) + **토스 송금/카카오 송금 링크**(국내) 둘 다.
- 카피: “이 도구가 마음에 들면 커피 한 잔 ☕” — 결과 복사 토스트 뒤 1회/세션 살짝 노출(선택).

## 2. AdSense는 ‘콘텐츠’에 — 승인·단가의 현실
- **도구 페이지(SPA)** 는 AdSense에 약하다: 페이지 전환이 적고 텍스트가 빈약해 *승인*도 *단가*도 낮다.
- 강한 자리는 **ddukkit.com 홈의 ‘읽을거리’(블로그성 글)**. 예) “인스타 글씨체 깨짐 없이 쓰는 법”,
  “디스코드 ASCII 아트 정렬 안 깨지게”, “카톡 닉 특수문자 추천 100선” — *글꾸미로 자연 유입* 되는 SEO 글.
- 승인 단위는 **도메인(ddukkit.com)**. `/geulkkumi` 는 같은 도메인이라 **재승인 불필요**,
  `ads.txt` 도 **도메인 루트(ddukkit.com/ads.txt) 하나**면 하위 경로 전체 커버.

## 3. 도구 안에는 ‘딱 1개, 스크롤 끝’ 만
- 위치: **‘꾸미기’ 라이브러리 / 아스키 갤러리 맨 아래**(콘텐츠 다 본 뒤). 결과 리스트·복사 동선 위엔 절대 X.
- 형태: 반응형 인피드 1개. 지연 로드. 고정(스티키)·전면(interstitial) 금지(정책·이탈 위험).
- 구현은 **기본 OFF** (`js/ads.js`). 퍼블리셔 ID를 넣기 전엔 **DOM에 아무것도 추가되지 않는다.**

### 켜는 법 (승인·ID 확보 후)
`index.html` `<head>` 에 한 줄(본인 ID로 교체):
```html
<script>window.GEULKKUMI_ADS = { client: "ca-pub-XXXXXXXXXXXXXXXX", librarySlot: "1234567890" };</script>
```
- `client` 가 없으면 `ads.js` 의 모든 함수가 **무동작(no-op)** → 지금처럼 광고 0.
- `ads.txt`(도메인 루트)에 `google.com, pub-XXXXXXXXXXXXXXXX, DIRECT, f08c47fec0942fa0` 추가.

## 4. PRO(광고/워터마크 제거) — 성장 루프
- 무료 **PNG 내보내기에 작은 ‘글꾸미’ 워터마크** → 공유될 때마다 자연 홍보(획득 루프).
- **PRO(한 번 결제 or 저가 구독)**: 워터마크 제거 + 광고 제거 + 글꼴팩/테마 추가 + 고해상도 PNG + 슬롯 무제한.
- 가격 감각: 일회성 ₩2,900~4,900 또는 월 ₩1,500 수준(도구앱 충동결제 구간). 결제는 토스페이/Gumroad 등.

## 5. 테마·스티커팩 IAP (나중)
- 시즌 한정(크리스마스/벚꽃) **이모지 아트 팩·프레임 팩·테마**를 소액 판매. 원하는 사람만 → UX 비용 0.

## 6. 토스 미니앱 보상형 (입점 시)
- 광클대전이 쓰는 **앱인토스 광고**와 동일 생태계. 글꾸미를 토스 미니앱으로 올리면
  *자발적* 보상형(예: ‘PRO 글꼴팩 24시간 체험’ 대가로 광고 1회) 도입 가능 — 강제 아님.

---

## KPI·게이팅(언제 광고를 켜나)
- **켜기 전 조건**: 일 방문 *의미 있는 수준*(예: DAU 500+) + 콘텐츠 글 5편+ + AdSense 승인.
  그 전엔 광고를 켜도 수익은 푼돈인데 이탈만 늘린다 → **후원 + 콘텐츠 SEO 먼저.**
- **측정**: 복사 수(핵심 가치) / 방문 / PNG 저장 / 후원 클릭 / (켜면)광고 RPM·CTR.
- **광고 켠 뒤 가드레일**: 복사 완료율·재방문이 떨어지면 슬롯 축소/제거. *도구 경험 > 광고*.

## 정책 주의 (AdSense)
- 도구 자동생성 결과(특수문자·아트)만 있는 ‘빈약한’ 페이지엔 광고 X → 콘텐츠 페이지에.
- 클릭 유도·라벨 혼동·전면광고 남용 금지. 개인정보/쿠키 고지(콘텐츠 페이지 푸터) 필요.
- 미성년 트래픽 비중 고려(아동 대상 한정 시 제약) — 일반 이용 가정.

---

### 요약
지금: **후원 링크(1) + 콘텐츠 SEO 글로 AdSense(2)**. 트래픽 붙으면 **도구 하단 1슬롯(3)** 과
**PNG 워터마크→PRO(4)**. 도구 셸은 끝까지 광고-라이트로. `js/ads.js` 는 **ID 넣기 전까지 완전 OFF.**
