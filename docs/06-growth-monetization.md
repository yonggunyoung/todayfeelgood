# 06. 성장·유료화 실행 계획 (Go-to-Market)

> 2026-06 작성. 배포 채널 확정안 + 앱인토스 검토 + 홍보 플랜 + 유료화 타임라인.

## 1. 배포 채널 전략 (확정)

```
[지금]   GitHub Pages(앱 본체) + 뚝딱(랜딩·버튼) + 도메인 연결(app.OOO.kr)
   ↓ 지표 검증 (북극성: 주 2회 요리완료, D7 ≥ 25%)
[2단계]  서버 경유 AI(Functions) + 계정 로그인 + 웹 구독 결제
   ↓
[3단계]  앱인토스 미니앱 출시  ←  한국 유통 1순위 (스토어보다 먼저)
   ↓
[4단계]  Capacitor 래핑 → 플레이스토어/앱스토어
```

## 2. 앱인토스(토스 미니앱) 검토 — 결론: 3단계 핵심 채널

**달라진 사실**: 토스의 미니앱 플랫폼 "앱인토스"는 2025년 오픈 후 **계약 없이 누구나(1인 개발자 포함) 출시 가능**한 오픈 플랫폼으로 전환됐고, 2026년 4월 기준 미니앱 2,000개를 돌파했다. 평균 출시 소요 ~5일 사례가 홍보될 만큼 진입장벽이 낮다.

**우리에게 좋은 이유**
- 토스 MAU 수천만의 유통 — 앱 설치 장벽 없이 토스 안에서 바로 실행
- 결제가 토스 안에 내장 → 웹 결제/IAP 수수료 고민의 상당 부분 해소 (수수료율은 입점 시 약관 확인)
- 우리 앱은 순수 웹 기반이라 미니앱(웹뷰+SDK) 포팅 난도가 낮은 편

**선행 조건 (그래서 3단계)**
1. **서버 경유 AI 필수** — 토스 사용자에게 "API 키 입력"을 시킬 수 없음. 2단계가 끝나야 입점 가능
2. 앱인토스 SDK/디자인 가이드 적응 (공식 문서 확인 필요: toss.im/apps-in-toss)
3. 수익 정산용 사업자등록
4. 콘텐츠 심사 대비: 산모 모드 의료 고지, AI 결과 신고 버튼

**액션**: 2단계 진입 시 앱인토스 개발자 문서를 검토해 포팅 공수 산정(예상 1~2주).

## 3. 홍보 플랜 (예산 0원 기준)

### 핵심 소재 — 우리 앱의 "15초 와우"
1. 영수증 사진 1장 → 냉장고가 채워지는 장면 (AI 스캔)
2. 냉기가 흐르는 냉장고 안에서 재료 꺼내 탭 → 추천이 바뀌는 장면
3. 유튜브 링크 붙여넣기 → 영상 안 보고 레시피가 정리되는 장면
4. "이번 달 아낀 돈 ₩31,000" 절약 장부

→ 이 4개를 각각 15~30초 세로 영상으로 제작 (인스타 릴스/유튜브 쇼츠/틱톡 동시 업로드)

### 단계별 채널

| 단계 | 채널 | 메시지 | 목표 |
|---|---|---|---|
| 베타 (지금~) | 지인 카톡, 가족 단톡 | "냉장고 사진 찍어봐" | 사용자 10~20명, 버그·UX 피드백 |
| 오가닉 1 | 메이커 커뮤니티 (디스콰이엇, GeekNews) | 만든 과정 + 데모 | 초기 피드백, 동료 메이커 확산 |
| 오가닉 2 | 숏폼 (릴스/쇼츠/틱톡) 주 2~3개 | 와우 장면 4종 반복 변주 | 타겟별 도달 (자취·운동·주부) |
| 오가닉 3 | 커뮤니티 침투 — 자취(에브리타임·오픈채팅), 운동(헬스 커뮤니티), 주부(맘카페·더쿠) | 모드별 맞춤 소구: "냉파 모드", "단백질 모드" | 모드 = 채널별 광고 카피 그 자체 |
| 검색 | 네이버 블로그 2~3편: "냉장고 파먹기 앱", "냉장고 관리 어플 추천" | 사용기 형태 | 검색 유입 |
| 제휴 (지표 후) | 요리 유튜버 — "이 영상 레시피, 냉비서에 저장하세요" 설명란 링크 | 유튜브 저장 기능과 직결되는 윈윈 | 크리에이터당 수백~수천 유입 |

### 준비물 체크리스트
- [ ] 앱 이름 확정 + 상표 검색(키프리스) + 도메인 구매
- [ ] 뚝딱 랜딩: 한 줄 가치 제안 + 데모 영상 + "앱 시작" 버튼
- [ ] 스크린샷 5장, 데모 영상 4종
- [ ] 인스타 계정 (제작 과정/팁 콘텐츠 겸용)

## 4. 유료화 실행 계획

### 단계 게이트 (숫자가 열쇠)

| 게이트 | 조건 | 하는 일 |
|---|---|---|
| G0 (지금) | — | 무료 + BYOK(고급 설정). 쿠팡 파트너스 **지금 신청** (승인 리드타임) |
| G1 | 주간 요리완료 ≥ 2회/사용자, D7 ≥ 25% | 서버 경유 AI 구축, 구글 로그인, 무료 월 10회 한도 |
| G2 | MAU 500~1,000 | 프리미엄 출시: 월 3,900원/연 29,000원 (A/B: 2,900/4,900) — 웹 결제(토스페이먼츠/포트원) |
| G3 | MAU 5,000+ | 앱인토스 입점(토스 결제), 스토어 출시, 크리에이터 제휴 |

### 수익원 우선순위
1. **쿠팡 파트너스** (G0부터) — 장보기 링크 커미션 ~3%, 구독 무관 전 사용자 대상
2. **프리미엄 구독** (G2) — AI 무제한·모드 무제한·가족 공유·식단 플랜
3. **앱인토스 내 결제** (G3) — 토스 트래픽용 동일 구독
4. (후순위) 크리에이터·커머스 제휴

### 행정 준비 (G1~G2 사이)
- 사업자등록 (간이과세 시작 가능) → 통신판매업 신고
- 개인정보처리방침 페이지 (현행 로컬 저장 구조는 부담 적음 — 서버 도입 시 갱신)
- 결제 PG 가입 (토스페이먼츠 등 — 사업자 필요)

### 비용 가드레일
- G1 전까지 고정비 0원 유지 (현재 구조)
- G1 후 AI 변동비 상한 알람 (일 $20) — `docs/04` 참고
- 무료 한도는 "도달률 25~40%" 구간으로 튜닝 (`docs/03`)

## 5. 이번 주 액션 (요약)

1. 도메인 구매 → CNAME 연결 (10분)
2. 쿠팡 파트너스 가입 신청
3. 뚝딱 랜딩 정리 + 버튼 연결
4. 지인 10명 배포 → 피드백 수집 시작
5. 와우 장면 1번(영수증 스캔) 숏폼 1개 제작·업로드
