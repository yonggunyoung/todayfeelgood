# ✦ 글꾸미 (geulkkumi)

**텍스트를 멋글씨로, 사진·그림을 도트·문자 아트로 — 복사해서 인스타·카톡·디스코드·게임 어디든 붙여넣기.**
[snskeyboard.com/instafont](https://snskeyboard.com/instafont.php) 류의 특수문자 변환기를 출발점으로, **사진/그리기 → 아트**, **혼합 생성기**, **대형 특수문자·이모티콘·텍대 DB**, 그리고 **휴대폰 앱(PWA: 홈 화면 설치·앱 단축키·공유로 받아 변환)** 까지 한 앱에 담았습니다.

빌드 과정이 없는 **정적 웹앱(PWA)** 이라 어떤 정적 호스팅에든 올리면 바로 동작하고, 휴대폰에서는 홈 화면에 설치해 앱처럼 씁니다.

## 기능 (4기둥 + α)

| 화면 | 기능 |
|---|---|
| ✨ **멋글씨** | 입력 → 30+ 유니코드 스타일(볼드·필기체·프락투어·더블스트럭·버블·네모·전각·작은대문자·뒤집기·취소선…) 실시간 변환 + 한글 **자모 분해·초성체·데코**. 탭하면 복사. |
| 🖼️ **사진아트** | 사진 업로드/드래그/붙여넣기 → **도트(브라유 2×4)·문자(ASCII)·하프블록·블록 음영·이모지** 아트. 폭·반전·디더·명암·문자셋 조절, **PNG 저장**. |
| ✏️ **그리기** | 손으로 그린 그림을 도트·문자 아트로 변환. 펜 굵기·지우개·해상도. |
| 🎀 **꾸미기 DB** | 특수문자(20+ 카테고리)·카오모지(감정별)·**혼합 생성기**(글꼴＋끼우기＋프레임)·구분선·**텍대 블록**. 검색·즐겨찾기. |
| 🗂️ **보관함** | 즐겨찾기 ★ · 최근 복사 기록. 어디서든 ☆ 한 번으로 모아두기. |

**휴대폰 편의 (PWA)**
- 홈 화면에 **설치**하면 전체화면 단독 앱 + 오프라인 동작(서비스워커 캐시).
- **앱 단축키**(길게 눌러 바로가기): 멋글씨 / 사진아트 / 그리기 / 꾸미기.
- **공유 타깃**: 다른 앱에서 텍스트를 *공유*하면 글꾸미가 열리며 변환 화면에 자동 입력 → 변환 → 복사.

## 바로 띄우기 (GitHub Pages)

1. 이 브랜치를 `main`에 병합 → Settings → Pages → Source: `main` / `/(root)`
2. 발급 주소: `https://yonggunyoung.github.io/todayfeelgood/geulkkumi/`

Vercel·Netlify·Cloudflare Pages도 저장소 연결만 하면 끝(빌드 명령 없음, 출력 = 폴더 루트). 독립 도메인을 쓰면 `CNAME` 추가 + AdSense `ads.txt`를 도메인 루트에 두면 됩니다.

## 개발

```bash
cd geulkkumi
npm start   # http://localhost:8080 (python3 http.server)
npm test    # 변환·아트·꾸미기·한글 코어 단위 테스트 (node --test, 25케이스)
```

빌드 도구·런타임 의존성 0. 구조:

```
index.html            앱 셸(헤더·탭바·공유메타)
css/styles.css        디자인 시스템(파스텔 Y2K, 라이트/다크)
js/main.js            라우팅·공유타깃·테마·PWA 설치
js/store.js           상태(즐겨찾기·최근·설정) + localStorage
js/ui.js · png.js     DOM·토스트·복사/공유 헬퍼, 아트 PNG 저장
js/engine/            순수 코어(무 DOM, 테스트 대상)
  unicode-fonts.js      유니코드 멋글씨 변환(예외표 포함)
  decorate.js           프레임·끼우기·혼합
  hangul.js             한글 자모/초성/데코
  ascii-art.js          밝기그리드 → 도트/문자/블록/이모지
js/data/              대형 DB(특수문자·카오모지·프레임·텍대·램프)

tests/                코어 단위 테스트
```

## 수익화(준비됨)

- **후원 링크**(헤더 ☕) — 네트워크 광고 없이 즉시 가능.
- **디스플레이 광고**: 독립 도메인 연결 후 AdSense `ads.txt`를 도메인 루트에 두고 셸 하단 슬롯에 게재.
- **Pro(선택)**: 광고 제거 · 대용량 사진 고해상 변환 · 프리미엄 프레임/팔레트 팩.
- 모든 처리는 **클라이언트 단독**(사진/그림이 서버로 가지 않음) → 개인정보·서버비용 0.
